Changelog format:
```
## [Unreleased]
### Added — new features
### Changed — modifications
### Fixed — bug fixes
```

API docs: table with Endpoint | Method | Auth | Description
Component docs: table with Component | Props | Description
Use imperative mood. One line per change. Reference file:line.
